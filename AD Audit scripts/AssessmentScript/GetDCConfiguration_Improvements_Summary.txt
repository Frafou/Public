USAGE EXAMPLES
==============

Basic Usage (Backward Compatible):
----------------------------------
Original VBScript:
    cscript GetDCConfiguration.vbs "DC=contoso,DC=com"

New PowerShell:
    .\GetDCConfiguration_v2.0.ps1 -DomainDN "DC=contoso,DC=com"

Advanced Enterprise Usage:
--------------------------
Security Assessment:
    .\GetDCConfiguration_v2.0.ps1 -DomainDN "DC=corp,DC=com" -IncludeSecurityAssessment -ComplianceReport -OutputFormat Excel,HTML

Multiple Output Formats:
    .\GetDCConfiguration_v2.0.ps1 -DomainDN "DC=test,DC=com" -OutputFormat CSV,JSON,Excel -OutputPath "C:\Reports"

Automation Integration:
    .\GetDCConfiguration_v2.0.ps1 -DomainDN "DC=prod,DC=com" -Silent -OutputFormat JSON

Quick Assessment:
    .\GetDCConfiguration_v2.0.ps1 -DomainDN "DC=dev,DC=com" -ExcludeSoftware -ExcludeFeatures

TECHNICAL SPECIFICATIONS
========================

Requirements:
? PowerShell 5.1 or higher
? Active Directory PowerShell module
? Administrative privileges (recommended)
? Network connectivity to Domain Controllers
? ImportExcel module (optional, for Excel output)

New Parameters:
- DomainDN (mandatory): Domain Distinguished Name
- OutputPath: Directory for output files
- OutputFormat: Console, CSV, JSON, Excel, XML, HTML
- CredentialPath: Encrypted credential file path
- TimeoutSeconds: Connection timeout (15-600 seconds)
- IncludeSecurityAssessment: Enable security analysis
- ComplianceReport: Generate compliance reports
- IncludeHealthCheck: DC health monitoring
- EmailReport: Send results via email
- Silent: Automation mode
- ConfigFile: Configuration file path
- ExcludeSoftware: Skip software inventory
- ExcludeFeatures: Skip server features

Output Files Generated:
- {Domain}_DNSConfiguration_{timestamp}.csv
- {Domain}_SoftwareInstalled_{timestamp}.csv
- {Domain}_ServerFeatures_{timestamp}.csv
- {Domain}_DCConfiguration_{timestamp}.json
- {Domain}_DCConfiguration_{timestamp}.xlsx
- {Domain}_DCConfiguration_{timestamp}.html
- DCConfiguration_{timestamp}.log
